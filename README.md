# Política de Segurança da Informação - NexaCommerce

## Sobre o projeto
Este projeto tem como objetivo desenvolver uma Política de Segurança da Informação (PSI) para uma empresa fictícia chamada NexaCommerce, que atua no setor de comércio eletrônico.

A empresa lida com dados sensíveis de clientes e transações financeiras, sendo necessária a criação de diretrizes para proteção dessas informações.

## Empresa fictícia
NexaCommerce é uma empresa de e-commerce especializada na venda de produtos eletrônicos e acessórios pela internet.

Setor: Comércio eletrônico  
Porte: Grande empresa

## Sistemas e dados críticos
- Banco de dados de clientes
- Sistema de pagamentos online
- Sistema de gestão de pedidos
- Servidor da plataforma de e-commerce
- Sistema administrativo interno
